и§ӮзңӢзҢҙеҚҡеЈ«зҲұи®ІиҜҫ-жҰӮзҺҮи®әдёҺж•°зҗҶз»ҹи®Ўи§Ҷйў‘ж—¶пјҢеӣ дёәи§Ҷйў‘дёҚеҲ©дәҺжҗңзҙўпјҢжүҖд»ҘеҒҡдёӘиҜҰз»ҶзӮ№зҡ„еҗ¬иҜҫз¬”и®°

еңЁжӣҙж–°е®ҢеҗҺпјҢеҰӮжһңйңҖиҰҒmdж–Үд»¶зҡ„еҸҜд»ҘеңЁж–Үжң«жҹҘзңӢе…ій”®иҜҚгҖӮ

дёҖгҖҒдәӢд»¶зҡ„жҰӮзҺҮ

1.ж— ж”ҫеӣһзұ»йўҳзӣ®

$$P = \frac{c ^{жқЎд»¶дёҖеҸ–}{жқЎд»¶дёҖжҖ»} \times c ^{жқЎд»¶дёҖеҸ–}{жқЎд»¶дёҖжҖ»} \times вҖҰ \times c ^{жқЎд»¶дёҖеҸ–}_{жқЎд»¶дёҖжҖ»}}{c^еҸ–_жҖ»}$$

$$C^m_n = \frac{n!}{m! \cdot (n-m)}$$

пјҲ1пјүзӣ’еӯҗдёӯжңү4зәў3зҷҪе…ұ7дёӘзҗғпјҢдёҚз”Ёзңјзһ…пјҢ7дёӘзҗғж‘ёиө·жқҘдёҖж ·пјҢзҺ°ж— ж”ҫеӣһзҡ„ж‘ё4ж¬ЎпјҢйӮЈж‘ёеҮәдёӨдёӘзәўзҗғдёӨдёӘзҷҪзҗғзҡ„жҰӮзҺҮжҳҜеӨҡе°‘пјҹ

$$P=\frac{C^2_4 \times C^2_3} {C^4_7}$$

пјҲ2пјүйҡ”еЈҒеұұеӨҙе…ұжңү11еҸӘжҜҚзҢҙе„ҝпјҢе…¶дёӯжңү5еҸӘзҫҺзҢҙе„ҝгҖҒ6еҸӘдё‘зҢҙе„ҝпјҢеңЁеӨ§й»‘еӨ©зңӢиө·жқҘжҳҜдёҖж ·зҡ„гҖӮд»Ҡе„ҝжңҲй»‘йЈҺй«ҳпјҢжҲ‘е°ҸејҹеҶ’жӯ»дёәжҲ‘жҚһжқҘ5еҸӘпјҢй—®еӨ©дә®еҗҺпјҢеҸ‘зҺ°жңү2еҸӘзҫҺзҢҙе„ҝгҖҒ3еҸӘдё‘зҢҙе„ҝзҡ„жҰӮзҺҮжҳҜеӨҡе°‘пјҹ

$$P=\frac{C^2_5 \times C^3_6} {C^5_{11}}=\frac{100}{231}$$

2.жңүж”ҫеӣһзұ»йўҳзӣ®

Kз§ҚйўңиүІзҡ„зҗғпјҢд»ЈеҸ·еҲҶеҲ«дёә$A_1гҖҒA_zвҖҰA_k$

жҠҪдёҖж¬ЎпјҢеҮәзҺ°зҡ„жҰӮзҺҮеҲҶеҲ«дёә$p_1гҖҒp_2вҖҰp_k$

жұӮж‘ёеҮәеҗ„з§Қзҗғзҡ„дёӘж•°еҲҶеҲ«дёә$n_1гҖҒn_zвҖҰn_k$

$p = \frac{(n_1 + n_2 + вҖҰ + n_k)!}{n_1!n_2!вҖҰn_k!}P_1^{n_1}P_2^{n_2}вҖҰP_k^{n_k}$

пјҲ1пјүзӣ’еӯҗдёӯжңү5зәў6зҷҪе…ұ11дёӘзҗғпјҢдёҚз”Ёзңјзһ…пјҢ11дёӘзҗғж‘ёиө·жқҘжҳҜдёҖж ·зҡ„пјҢзҺ°жңүж”ҫеӣһзҡ„ж‘ё5ж¬ЎпјҢйӮЈж‘ёеҮәдёӨдёӘзәўзҗғдёүдёӘзҷҪзҗғзҡ„жҰӮзҺҮжҳҜеӨҡе°‘пјҹ

и§Јпјҡ

2з§ҚйўңиүІзҡ„зҗғпјҢд»ЈеҸ·еҲҶеҲ«дёәзәўгҖҒзҷҪ

жҠҪдёҖж¬ЎпјҢеҮәзҺ°зҡ„жҰӮзҺҮеҲҶеҲ«дёә$\frac{5}{11}гҖҒ\frac{6}{11}$

жұӮж‘ёеҮәеҗ„з§Қзҗғзҡ„дёӘж•°еҲҶеҲ«дёә2гҖҒ3

$P = \frac{(2+3)!}{2!3!}(\frac{5}{11})^2(\frac{6}{11})^3$

пјҲ2пјүеңЁе°ҸејҹдёәжҲ‘жҠ“еӣһзҡ„5еҸӘжҜҚзҢҙе„ҝдёӯпјҢжңү2зҫҺ3дё‘пјҢжҜҸеӨ©жҲ‘йғҪйҡҸжңәжҢ‘дёҖеҸӘжҜҚзҢҙе„ҝжқҘпјҢдёәеҘ№жҠ“иҷұеӯҗгҖӮ
е°ұиҝҷж ·пјҢиҝҮеҺ»дәҶ101еӨ©пјҢжҠ“дәҶ101ж¬ЎиҷұеӯҗпјҢй—®иҝҷ101ж¬ЎдёӯпјҢдёәзҫҺзҢҙе„ҝжңҚеҠЎ50ж¬ЎгҖҒдё‘зҢҙе„ҝжңҚеҠЎ51ж¬Ўзҡ„жҰӮзҺҮжҳҜеӨҡе°‘пјҹ

и§Јпјҡ

2з§ҚжҜҚзҢҙе„ҝпјҢд»ЈеҸ·еҲҶеҲ«дёәзҫҺгҖҒдё‘

йҖүдёҖеҸӘпјҢеҮәзҺ°жҰӮзҺҮеҲҶеҲ«дёә$\frac{2}{5}гҖҒ\frac{3}{5}$

дёә2з§ҚжңҚеҠЎж¬Ўж•°еҲҶеҲ«дёә50гҖҒ51

$P = \frac{(50+51)!}{50!51!} (\frac{2}{5})^{50} (\frac{3}{5})^{51}$

3.йңҖиҰҒз”»еӣҫзҡ„йўҳзӣ®

пјҲ1пјүе·ІзҹҘ0<x<1пјҢ0<y-1пјҢжұӮx>yзҡ„жҰӮзҺҮжҳҜеӨҡе°‘пјҹ

в‘ иЎЁзҺ°е·ІзҹҘжқЎд»¶

в‘ЎиЎЁзҺ°еҫ…жұӮжҰӮзҺҮзҡ„жқЎд»¶

в‘ўжүҫеҮәв‘ в‘ЎйҮҚеҗҲйғЁеҲҶ

в‘Ј$P(x>y)=\frac{в‘ў}{в‘ }$

пјҲ2пјүе·ІзҹҘе°ҸжҳҺдјҡеңЁ0зӮ№д№ӢеҗҺ1зӮ№д№ӢеүҚеҲ°ж•ҷе®ӨпјҢе°ҸеҲҡд№ҹжҳҜпјҢй—®е°ҸжҳҺжҜ”е°ҸеҲҡжҷҡеҲ°зҡ„жҰӮзҺҮжҳҜеӨҡе°‘пјҹ

и®ҫе°ҸжҳҺеҲ°ж•ҷе®Өж—¶й—ҙдёәxпјҢе°ҸеҲҡдёәy

е·ІзҹҘ0<x<1пјҢ0<y<1пјҢжұӮx>yзҡ„жҰӮзҺҮжҳҜеӨҡе°‘пјҹ

пјҲ3пјүе·ІзҹҘ-1<x<1пјҢ-1<y<1пјҢжұӮ
$$x^2+y^2<1$$
зҡ„жҰӮзҺҮжҳҜеӨҡе°‘пјҹ


и§Јжһҗпјҡ

в‘ иЎЁзҺ°е·ІзҹҘжқЎд»¶

в‘ЎиЎЁзҺ°еҫ…жұӮжҰӮзҺҮзҡ„жқЎд»¶

в‘ўжүҫеҮәв‘ в‘ЎйҮҚеҗҲйғЁеҲҶ

в‘Ј$P(x^2 + y^2 < 1) = \frac { в‘ў }{ в‘  }$

и§Јпјҡ
$$S_еӣӯ = ПҖ \times 1^2 = ПҖ$$

$$S_жӯЈ = 4$$

$$P(x^2+y^2<1) = \frac{ПҖ}{4}$$

4.жқЎд»¶жҰӮзҺҮ

$$P(B \mid A) = \frac{ P(AB) }{ P(A) }$$

дәӢд»¶AпјҡжҺ·дёҖж¬ЎйӘ°еӯҗпјҢжңқдёҠзӮ№ж•°еӨ§дәҺ3

дәӢд»¶BпјҡжҺ·дёҖж¬ЎйӘ°еӯҗпјҢжңқдёҠзӮ№ж•°жҳҜ6

PпјҲB|AпјүпјҡжҺ·дёҖж¬ЎйӘ°еӯҗпјҢе·ІзҹҘжңқдёҠзӮ№ж•°еӨ§дәҺ3пјҢжңқдёҠзӮ№ж•°жҳҜ6зҡ„жҰӮзҺҮ

PпјҲABпјүпјҡжҺ·дёҖж¬ЎйӘ°еӯҗпјҢжңқдёҠзӮ№ж•°жҳҜ6зҡ„жҰӮзҺҮ

PпјҲAпјүпјҡжҺ·дёҖж¬ЎйӘ°еӯҗпјҢжңқдёҠзӮ№ж•°еӨ§дәҺ3зҡ„жҰӮзҺҮ

пјҲ1пјүе°ҸжҳҺжҰӮзҺҮи®әиҖғиҜ•еҫ—80еҲҶд»ҘдёҠзҡ„жҰӮзҺҮжҳҜ80%пјҢеҫ—60еҲҶд»ҘдёҠзҡ„жҰӮзҺҮжҳҜ85%пјҢе·ІзҹҘиҝҷж¬ЎиҖғиҜ•е°ҸжҳҺжҰӮзҺҮи®әжІЎжҢӮпјҢйӮЈд№Ҳе°ҸжҳҺеҫ—80еҲҶд»ҘдёҠзҡ„жҰӮзҺҮжҳҜеӨҡе°‘пјҹ

дәӢд»¶Aпјҡе°ҸжҳҺеҫ—60еҲҶд»ҘдёҠ

дәӢд»¶Bпјҡе°ҸжҳҺеҫ—80еҲҶд»ҘдёҠ

PпјҲB|Aпјүпјҡе°ҸжҳҺеҫ—60еҲҶд»ҘдёҠж—¶пјҢе°ҸжҳҺеҫ—80еҲҶд»ҘдёҠзҡ„жҰӮзҺҮпјҢеҚіе°ҸжҳҺеҫ—80еҲҶд»ҘдёҠзҡ„жҰӮзҺҮ

$$P(B \mid A) = \frac{ P(AB) }{ P(A) } = \frac{ 0.8 }{ 0.85 }$$

пјҲ2пјүжҹҗең°еҢәд»Ҡе№ҙдјҡеҸ‘з”ҹжҙӘж°ҙзҡ„жҰӮзҺҮжҳҜ80%пјҢд»ҠжҳҺдёӨе№ҙиҮіе°‘жңүдёҖе№ҙдјҡеҸ‘з”ҹжҙӘж°ҙзҡ„жҰӮзҺҮжҳҜ85%пјҢеҒҮеҰӮд»Ҡе№ҙжІЎжңүеҸ‘з”ҹжҙӘж°ҙпјҢйӮЈд№ҲжҳҺе№ҙеҸ‘з”ҹжҙӘж°ҙзҡ„жҰӮзҺҮжҳҜеӨҡе°‘пјҹ

дәӢд»¶Aпјҡд»Ҡе№ҙжІЎжңүеҸ‘з”ҹжҙӘж°ҙ

дәӢд»¶BпјҡжҳҺе№ҙеҸ‘з”ҹжҙӘж°ҙ

PпјҲBIAпјүпјҡд»Ҡе№ҙжІЎжңүеҸ‘з”ҹжҙӘж°ҙзҡ„жғ…еҶөдёӢпјҢжҳҺе№ҙеҸ‘жҙӘж°ҙзҡ„жҰӮзҺҮ

$$P(B \mid A) = \frac{ P(AB) }{ P(A) } = \frac{ 0.05 }{ 0.2 }$$

5.е…ЁжҰӮзҺҮе…¬ејҸ

AгҖҒBвҖҰзӯүдёӘдҪ“еқҮеҸҜиғҪеҸ‘з”ҹжҹҗдәӢпјҢеҲҷ

$$P(еҸ‘з”ҹжҹҗдәӢ)=P(AеҮәзҺ°)В·P(AеҸ‘з”ҹжҹҗдәӢ)+P(BеҮәзҺ°)В·P(BеҸ‘з”ҹжҹҗдәӢ)вҖҰ$$

пјҲ1пјүжҹҗй«ҳйҖҹе…¬и·ҜдёҠе®ўиҪҰдёӯжңү20%жҳҜй«ҳйҖҹе®ўиҪҰпјҢ80%жҳҜжҷ®йҖҡе®ўиҪҰпјҢеҒҮи®ҫй«ҳйҖҹе®ўиҪҰеҸ‘з”ҹж•…йҡңзҡ„жҰӮзҺҮжҳҜ0.002пјҢжҷ®йҖҡе®ўиҪҰеҸ‘з”ҹж•…йҡңзҡ„жҰӮзҺҮжҳҜ0.01гҖӮжұӮиҜҘй«ҳйҖҹе…¬и·ҜдёҠжңүе®ўиҪҰеҸ‘з”ҹж•…йҡңзҡ„жҰӮзҺҮгҖӮ

$$P(жңүе®ўиҪҰеҸ‘з”ҹж•…йҡң)=P(й«ҳйҖҹе®ўиҪҰеҮәзҺ°)В·P(й«ҳйҖҹе®ўиҪҰж•…йҡң)+P(жҷ®йҖҡе®ўиҪҰеҮәзҺ°)В·P(жҷ®йҖҡе®ўиҪҰж•…йҡң)$$

$P(жңүе®ўиҪҰеҸ‘з”ҹж•…йҡң)=0.2В·0.002+0.8В·0.01=0.0084$

(2)зҢҙеҚҡеЈ«е…¬еҸёжңүзҢҙеҚҡеЈ«дёҺеӮ»еӯўеӯҗдёӨдёӘе‘ҳе·ҘпјҢиҖҒжқҝиҰҒжҠҪе…¶дёӯдёҖдёӘиҖғж ёпјҢжҠҪдёӯзҢҙеҚҡеЈ«дёҺеӮ»еӯўеӯҗзҡ„жҰӮзҺҮйғҪжҳҜ50%пјҢзҢҙеҚҡеЈ«иҖғж ёйҖҡиҝҮзҡ„жҰӮзҺҮжҳҜ100%пјҢеӮ»еӯўеӯҗиҖғж ёйҖҡиҝҮзҡ„жҰӮзҺҮжҳҜ1%пјҢйӮЈд№ҲжҠҪдёӯзҡ„е‘ҳе·ҘйҖҡиҝҮиҖғж ёзҡ„жҰӮзҺҮжҳҜеӨҡе°‘пјҹ

$$PпјҲжҠҪдёӯзҡ„е‘ҳе·ҘйҖҡиҝҮиҖғж ёпјү=PпјҲзҢҙеҚҡеЈ«еҮәзҺ°пјүВ·PзҢҙеҚҡеЈ«йҖҡиҝҮпјү+PпјҲеӮ»еӯўеӯҗеҮәзҺ°пјүВ·PпјҲеӮ»зҠҜеӯҗйҖҡиҝҮпјү$$

$PпјҲжҠҪдёӯзҡ„е‘ҳе·ҘйҖҡиҝҮиҖғж ёпјү=50%В·100%+50%В·1%=50.5%$

пјҲ3пјүеҸҲеҲ°дәҶдәӨй…Қзҡ„еӯЈиҠӮпјҢдё»дәәиҝҮдёӨеӨ©е°ұжӢүдҪ еҺ»йҡ”еЈҒжқ‘жүҫжҜҚй©ҙй…Қз§ҚпјҢйҡ”еЈҒжқ‘жңүдёүеӨҙжҜҚй©ҙпјҢеҲҶеҲ«жҳҜзҷҪй©ҙгҖҒй»‘й©ҙе’ҢжЈ•й©ҙгҖӮеҘ№д»Ғеҫ—дҪ е® е№ёзҡ„жҰӮзҺҮеҲҶеҲ«жҳҜ50%гҖҒ20%гҖҒ30%гҖӮе°ҸзҷҪеұҒиӮЎеӨ§иғҪз”ҹпјҢеҘ№иғҪжҖҖдёҠдҪ еӯ©еӯҗзҡ„жҰӮзҺҮжҳҜ80%гҖӮе°Ҹй»‘еӨӘзҳҰе°ҸпјҢиғҪжҖҖеӯ•зҡ„жҰӮзҺҮжҳҜ20%гҖӮе°ҸжЈ•дёӯи§„дёӯзҹ©пјҢиғҪдёӢжҒҜзҡ„жҰӮзҺҮжҳҜ50%гҖӮйӮЈд№ҲдҪ иғҪе–ңеҪ“зҲ№зҡ„жҰӮзҺҮжҳҜеӨҡе°‘пјҹ

AгҖҒ0% BгҖҒ59% CгҖҒ47%

6.иҙқеҸ¶ж–Ҝе…¬ејҸ

AгҖҒBвҖҰзӯүдёӘдҪ“еқҮеҸҜиғҪеҸ‘з”ҹжҹҗдәӢпјҢеҲҷ

$$P(е·ІзҹҘжңүдёӘдҪ“еҸ‘з”ҹжҹҗдәӢж—¶пјҢжҳҜAеҸ‘з”ҹзҡ„)= \frac{ P(AеҮәзҺ°) \cdot P(AеҸ‘з”ҹжҹҗдәӢ) }{ P(еҸ‘з”ҹжҹҗдәӢ) } $$

пјҲ1пјүжҹҗй«ҳйҖҹе…¬и·ҜдёҠе®ўиҪҰдёӯжңү20%жҳҜй«ҳйҖҹе®ўиҪҰпјҢ80%жҳҜжҷ®йҖҡе®ўиҪҰпјҢеҒҮи®ҫй«ҳйҖҹе®ўиҪҰеҸ‘з”ҹж•…йҡңзҡ„жҰӮзҺҮжҳҜ0.002пјҢжҷ®йҖҡе®ўиҪҰеҸ‘з”ҹж•…йҡңзҡ„жҰӮзҺҮжҳҜ0.01гҖӮжұӮиҜҘй«ҳйҖҹе…¬и·ҜдёҠжңүе®ўиҪҰеҸ‘з”ҹж•…йҡңж—¶пјҢж•…йҡңзҡ„жҳҜй«ҳйҖҹе®ўиҪҰзҡ„жҰӮзҺҮгҖӮ

$$PпјҲе·ІзҹҘжңүе®ўиҪҰеҸ‘з”ҹж•…йҡңпјҢжҳҜй«ҳйҖҹе®ўиҪҰеҸ‘з”ҹзҡ„пјү=\frac{0.2В·0.002}{0.0084}=\frac{1}{21}$$
