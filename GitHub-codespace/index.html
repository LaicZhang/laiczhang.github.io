Github-codespace简单使用

很早就有资格了，今天忽然想起来，就跑一跑。

GitHub Codespaces文档

创建

  1. 在repo的code界面，点击code,如果是在main分支进行创建，则可以直接点击Create codespace on main,否则需要自行选择Configure and create codespace
  2. 等待初始化完成即可在浏览器中使用
  3. 如需在本地vscode中使用，可以安装vscode desktop拓展
